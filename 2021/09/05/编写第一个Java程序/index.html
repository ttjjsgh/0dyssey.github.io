编写第一个Java程序

2021-09-05

一.定义类
在Java程序开发之中最基础的单元是类，所有的程序必须在类中执行，类的基本语法如下
：
[pubic] class 类名称
{
}

实例：
public class HelloWorld{
public static void main(String[] args){
System.out.println(“Hello，World!”);
}
}

注：“public class类名称{}”：类名称必须与文件名保持一致，一个Java*.java文件中只允许有一个public class定义；
Java语言有着明确的命名要求，以后定义类名称的时候要求每一个单词首字母大写，例如：HelloWorld

二.主方法
主方法是所有程序执行的起点，并且一定要定义在类中，Java的主方法定义
public class 类名称{
public static void main(String[] args){
程序代码由此执行;
}
}
屏幕打印（系统输出）
有两类语法形式
1.输出后追加换行：System.out.println();
2.输出后不追加换行：System.out.print();

源码分析：
public 表示公开
class 表示类
static 表示静态
void 表示方法返回值类型
main 表示方法名，严格定义
String 表示字符串
[] 表示数组
args 表示参数名
