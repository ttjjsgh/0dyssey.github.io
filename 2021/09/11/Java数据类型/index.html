Java数据类型

在Java中，数据类型分为基本数据类型和引用数据类型两大类。
基本数据类型有八种，分别是：
1.整型：byte、short、int、long
2.浮点型：float、double
3.字符型：char
4.布尔型：boolean

整型：
数据类型 所占比特位数 取值范围
byte 8(1个字节) -128127
short 16(2……) -3276832767
int 32(4……) -2的31次方2的31次方-1
long 64(8……) -2的63次方2的63次方-1

浮点型：
数据类型 所占比特位数
float 32(4个字节）
double 64(8个字节）

注：浮点数默认为double型，float型的数值有一个后缀“f“。

例如
float a=123.45f;
double b=123.45;

布尔类型：
boolean类型只有两个值：true和false，用于判定逻辑条件

例如：
Boolean a=false；
Boolean b=true；

字符型：
char：存放由单括号括起的单个字符，占两个字节，取值范围：[0,65535]

例如：char c=’中’;
char a=20013;

数据类型转换：
1.自动类型转换：把一个表示数据范围小的数值或者变量赋给另一个表示数据范围大的变量。这种转换方式是自动的，直接书写即可。
例如：double num=10;

2.强制类型转换：把一个表示数据范围大的数值或者变量赋值给数据范围小的变量。

3.强制类型转换格式：目标数据类型 变量名=(目标数据类型）值或者变量。

byte–>short
–>int–>long–>float–>double
